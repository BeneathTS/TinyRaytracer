# Tiny raytracer

Works on <u>MacOS 10.12 / 10.13</u>

### Install: 

`git clone https://github.com/BeneathTS/rtv1.git`

then

`make && make clean`

#### Launch:

`./RTv1 <SceneFolderPath/SceneName>`

For example: `./RTv1 scenes/scene1`
